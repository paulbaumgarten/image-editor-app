## Getting started

Cropping tool

 * Control-C ---> Start crop tool
 * Left/Right/Up/Down ---> Move cropping square within frame
 * +/- --> Enlarge/reduce cropping square
 * Return --> Lock in crop

Rotate

 * Control-R

Undo 

 * Control-Z

Next/previous image

 * Left/right arrow key
